# Code of Conduct

## Our Standards

This project is a technical space. Everyone who participates, whether by opening issues, commenting on pull requests, or contributing code, is expected to engage professionally and constructively.

Acceptable behavior includes :

- Providing clear, factual, and constructive feedback on code and ideas
- Welcoming contributions from people at all experience levels
- Focusing discussions on the technical merits of proposals
- Acknowledging mistakes openly and moving forward

Unacceptable behavior includes :

- Insults, personal attacks, or dismissive language toward any contributor
- Publishing private information (email addresses, usernames) without consent
- Sustained disruption of discussions or pull request reviews
- Any form of harassment, regardless of the channel

---

## Enforcement

Instances of unacceptable behavior may be reported by contacting the project maintainer through GitHub. All reports will be reviewed and investigated. Maintainers are obligated to maintain confidentiality with regard to the reporter.

Contributors who do not follow this Code of Conduct may be temporarily or permanently excluded from the project.

---

## Scope

This Code of Conduct applies within all project spaces, including GitHub issues, pull requests, and any other forum used by this project.
